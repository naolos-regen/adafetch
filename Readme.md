# Info-Linux-Sys

```bash
# Uptime-Info
>$ cat /proc/uptime

# OS-Info
>$ cat /etc/os-release
>$ cat /proc/version

# CPU-Info
>$ cat /proc/cpuinfo

# RAM-Info
>$ cat /proc/meminfo

# GPU-Info
>$ cat /sys/class/graphics/fb0/device/ device | vendor
>$ cat /sys/class/card/drm/card<n>/device/

# Battery-Info
>$ cat /sys/class/power_supply/BAT0

```
